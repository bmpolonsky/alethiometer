# Alethiometer Web

Browser port of the old Alethiometer mobile build.

## Stack

- Vite
- React
- TypeScript
- SVG overlay for dial interaction

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Notes

- The original Flash and AIR project files are kept untouched in `../Alethiometer Beta/`.
- Meanings, settings, and saved readings are stored in browser `localStorage`.
- The instrument art and fonts are reused from the original project archive, while the web UI and state model are rebuilt from scratch.
