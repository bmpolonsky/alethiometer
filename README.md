# Alethiometer

`Alethiometer` — веб-приложение, вдохновлённое алетиометром из мира `His Dark Materials`, символическим прибором, который отвечает не фразой, а знаками.

Здесь можно собрать вопрос из трёх символов, получить ответ в виде последовательности образов и трактовать его через базовые и личные значения.

## Что умеет приложение

- задавать вопрос тремя символами
- выбирать символы стрелками или через список
- показывать ответ четвёртой стрелкой прибора
- читать базовые значения каждого символа
- добавлять личные трактовки
- сохранять ответы в архив
- возвращаться к прежним ответам и заметкам
- открывать отдельную справку
- включать медитативный режим почти без интерфейса

## Как пользоваться

1. Выбери три символа, которые лучше всего описывают вопрос или ситуацию.
2. Нажми `Спросить`.
3. Дождись, пока четвёртая стрелка пройдёт по символам ответа.
4. Прочитай ответ, опираясь на базовые значения, личные трактовки и контекст вопроса.
5. Если ответ оказался важным, сохрани его в архив со своей формулировкой смысла.

## Что важно для трактовки

Алетиометр в приложении говорит образами, а не буквальными предложениями.

- Важен не только каждый символ, но и порядок их появления.
- Один и тот же знак может значить разное в разных вопросах.
- Со временем у тебя появляется собственный язык символов, поэтому в приложении есть личные трактовки.
- Базовые значения остаются опорой, а личные помогают сделать трактовку живой и по-настоящему своей.

## Что есть внутри

- раздел `Символы` со всеми 36 знаками
- отдельный просмотр значений выбранного символа
- редактирование личных трактовок
- архив ответов с вопросом, самим ответом и твоей формулировкой
- справка с кратким объяснением процесса и расширенными подсказками
